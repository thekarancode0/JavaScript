Alert

It shows a messasge and waits for the user to press "ok"

The visitor can't interact with rest of the page until they have dealt with the window "OK".

Prompt

The function Prompt accepts two args

promt(title,[default])---title: text to show.......default: initial value of input field

It shows a modelwindow with a text message, and input field and the buttons OK/cancel.

Confirm

This function shows a model window with a question and two buttons.
